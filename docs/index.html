Documentation

Important Prerequisite

AWSMGR operates as a wrapper around the official AWS CLI tool provided by Amazon. Please ensure that the AWS CLI is installed and properly configured on your system before using AWSMGR.

If you haven't installed AWS CLI yet, please install it and run the following command to configure it with the necessary permissions:

aws configure

This will prompt you to enter your AWS access key, secret key, default region, and output format. Make sure your AWS credentials have the appropriate permissions for the services you intend to manage.

Setting Up MFA Delete for S3 Buckets

To use the S3 MFA Delete feature, you need to configure your MFA device's ARN in AWSMGR:

  1. Find your MFA device ARN in the AWS Console:
    • Go to IAM → Users → Your Username → Security credentials
    • Under "Multi-factor authentication (MFA)", find your MFA device
    • Copy the device ARN (looks like: arn:aws:iam::123456789012:mfa/username)
  2. Save your MFA ARN to the AWSMGR config file:
    echo "YOUR_MFA_DEVICE_ARN" > ~/.config/awsmgr/aws_config/security_arn_mfa.txt

    Replace YOUR_MFA_DEVICE_ARN with your actual MFA device ARN from step 1.

After completing these steps, you can use the "Enable/Disable MFA Delete on Bucket" option in the S3 management menu.

Version 1.0.0 STABLE

March 2025

First stable release with significant improvements over the beta version.

  • Added MFA Delete for S3 Buckets Secure your S3 buckets with multi-factor authentication delete protection
  • IAM Password Management Set initial passwords and update existing passwords for IAM users
  • Redesigned User Interface Improved terminal UI with clearer navigation and visual feedback
  • Enhanced Animations Smoother transitions and loading indicators
  • Back Menu Navigation Added back menu option for all screens for easier navigation
  • Force Delete Operations Force delete buckets, users, and groups when needed
  • S3 Bugfixes Fixed several issues with S3 bucket management
  • Empty Value Handling Improved handling of empty buckets and empty string inputs
